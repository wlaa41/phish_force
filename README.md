# 🎮 ShadowPhish — Cyber Defense Academy

A browser-based cybersecurity awareness game where players learn to identify phishing, smishing, and social engineering attacks.

## 🕹️ Play Now
👉 **[Click here to play](https://wlaa41.github.io/phish_force/)**

## 🎯 Game Modes
- **Intel Mission** — Quiz-based phishing investigation
- **Highway Chase** — Endless bike chase with question checkpoints

## 🛠️ Built With
- HTML, CSS, JavaScript (no frameworks)
- Runs entirely in the browser — no install needed
